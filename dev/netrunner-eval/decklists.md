# Tutorial Decklists (System Gateway)

## Runner: The Catalyst

**Identity:** The Catalyst: Convention Breaker
- No special ability (tutorial ID)
- 40 card minimum, 15 influence

### Events (14)

**Creative Commission** (2x) - Cost 0
Gain 4 credits. You cannot play another copy of Creative Commission this turn.

**Jailbreak** (3x) - Cost 0, Run
Run HQ or R&D. If successful, draw 1 card and access 1 additional card.

**Overclock** (2x) - Cost 0, Run
Place 5 credits on this event, then run any server. When that run ends, trash Overclock. Use credits on Overclock during runs.

**Sure Gamble** (3x) - Cost 5
Gain 9 credits.

**Tread Lightly** (2x) - Cost 0, Run
Run any server. The first time the Corp rezzes a piece of ice during this run, they must pay 3 credits more.

**VRcation** (2x) - Cost 1
Draw 4 cards. If you have any clicks remaining, lose 1 click.

### Hardware (2)

**Docklands Pass** (1x) - Install 2
The first time you breach HQ each turn, access 1 additional card.

**Pennyshaver** (1x) - Install 3
Whenever you make a successful run, place 1 credit on this hardware.
Click: Take all credits from this hardware.

### Resources (6)

**Red Team** (1x) - Install 3
When you install this resource, load 12 credits onto it.
Click: Take 3 credits from this resource and run HQ, R&D, or Archives. When it is empty, trash it.

**Smartware Distributor** (2x) - Install 1
When your turn begins, place 1 credit on this resource.
Click: Take all credits from this resource.

**Telework Contract** (2x) - Install 1
When you install this resource, load 9 credits onto it.
Click: Take 3 credits from this resource. When it is empty, trash it.

**Verbal Plasticity** (1x) - Install 1
The first time you draw a card each turn (including your mandatory draw), draw 1 additional card.

### Icebreakers (8)

**Carmen** (2x) - Install 4, Strength 0, MU 1, Killer
Whenever this program fully breaks a piece of ice, place 1 power counter on Carmen.
Interface → 1 credit: Break 1 sentry subroutine.
Hosted power counter: +3 strength.

**Cleaver** (2x) - Install 3, Strength 1, MU 1, Fracter
Interface → 2 credits: Break up to 2 barrier subroutines.
2 credits: +1 strength.

**Mayfly** (2x) - Install 1, Strength 1, MU 1, All-types
Interface → 1 credit: Break 1 subroutine.
1 credit: +1 strength.
When this run ends, trash this program.

**Unity** (2x) - Install 1, Strength 1, MU 1, Decoder
Interface → 1 credit: Break 1 code gate subroutine.
1 credit: +1 strength for each installed icebreaker (including Unity).

## Corp: The Syndicate

**Identity:** The Syndicate: Profit over Principle
- No special ability (tutorial ID)
- 45 card minimum, 3 influence

### Agendas (7)

**Offworld Office** (3x) - Advancement 4, Points 2
When you score this agenda, gain 7 credits.

**Send a Message** (2x) - Advancement 3, Points 1
When this agenda is scored or stolen, you may rez 1 piece of ice, ignoring all costs.

**Superconducting Hub** (2x) - Advancement 3, Points 1
When you score this agenda, draw 2 cards.

### Assets (6)

**Nico Campaign** (2x) - Rez 0, Trash 3
When you rez this asset, load 9 credits onto it.
When your turn begins, take 3 credits from this asset. When it is empty, trash it.

**Regolith Mining License** (2x) - Rez 1, Trash 3
When you rez this asset, load 15 credits onto it.
Click: Take 3 credits from this asset. When it is empty, trash it.

**Urtica Cipher** (2x) - Rez 0, Trash 0, Ambush
If Urtica Cipher is accessed while installed, do 2 net damage plus 1 net damage for each advancement counter on it.

### Operations (7)

**Government Subsidy** (2x) - Cost 6
Gain 15 credits.

**Hedge Fund** (3x) - Cost 5
Gain 9 credits.

**Seamless Launch** (2x) - Cost 1
Place 2 advancement counters on 1 installed card that you did not install this turn.

### Upgrades (1)

**Manegarm Skunkworks** (1x) - Rez 2, Trash 4
Whenever the Runner approaches this server, they must either pay 2 credits and spend 1 click or end the run.

### ICE - Barriers (5)

**Brân 1.0** (2x) - Rez 6, Strength 6, Bioroid
Lose 1 click: Break 1 subroutine on this ice. Only the Runner can use this ability.
↳ The Runner cannot pass Brân 1.0 this run.
↳ Install 1 ice from HQ or Archives directly inward from Brân 1.0, unrezzed.
↳ End the run.

**Palisade** (3x) - Rez 3, Strength 2
Palisade gets +2 strength while protecting a remote server.
↳ End the run.

### ICE - Code Gates (4)

**Diviner** (2x) - Rez 2, Strength 2, AP
↳ Do 1 net damage.
↳ End the run.

**Whitespace** (2x) - Rez 2, Strength 3
↳ The Runner loses 3 credits. If the Runner has fewer than 6 credits, end the run.

### ICE - Sentries (4)

**Karunā** (2x) - Rez 4, Strength 3, AP
↳ Do 2 net damage. The Runner may jack out.
↳ Do 2 net damage.

**Tithe** (2x) - Rez 1, Strength 2
↳ Gain 1 credit.
↳ End the run.
